# 格闘技練習ノートアプリ

## 作った背景

趣味で総合格闘技をやっており、アマチュア大会への出場を目標にしている。
仕事の兼ね合いで頻繁に通うことが難しく、前回の練習内容を忘れがちだったため、
見返せる練習ノートを作ることにした。

既存のiOSアプリは丸印カレンダー形式で、前のジム（神戸）では
曜日ごとに「打撃・組み・寝技」がきれいに分かれていたため十分だった。
しかし東京転居後に通うジムは1日の中に複数カテゴリのクラスが混在しており、
「どのカテゴリの練習が足りていないか」を既存アプリでは分析しづらかった。

iPhone買い替えのコストやストレージ問題も考慮した結果、
自分の要件に合ったWebアプリを自作することにした。

## 機能

- 道場ごとの週間スケジュールをグリッド表示（HEARTS新宿 / KIX新宿 / KIX駒沢）
- 月曜始まりカレンダーで練習日・受講クラスを記録
- 打撃・柔術・組み・MMAの4カテゴリでスキルバランスをレーダーチャート可視化（月別/年別/累計）
- Gemini APIで「今日の意識」を練習前に1点提案（今日のクラス内容と過去の記録を加味）
- Gemini APIで目標に対する週間練習バランスと行動アドバイスを生成
- 過去のジム（神戸）時代の実績も入力・管理可能

## 技術構成

| 項目 | 内容 |
|------|------|
| フロントエンド | HTML / CSS / Vanilla JavaScript |
| AI連携 | Gemini API（gemini-2.5-flash-lite） |
| データ保存 | localStorage |
